Topics that must be covered

    What will be covered in this journey? (Deep Learning from perceptron to state-of-the-art architectures)

    The Perceptron (Mathematical neuron, activation functions — Sigmoid, Tanh, ReLU, Leaky ReLU, Swish, GELU)

    Forward Propagation (Matrix multiplication through layers)

    Backpropagation (Derivation of gradients using the chain rule)

    Optimizers (SGD, Momentum, Nesterov, RMSprop, Adam, Nadam, AdamW)

    Loss Functions (MSE for regression, Cross-Entropy for classification, Hinge Loss, Contrastive Loss)

    Regularization (L1/L2 weight decay, Dropout, Batch Normalization, Layer Normalization)

    Weight Initialization (Zero, Random, Xavier/Glorot, He/Kaiming, Orthogonal)

    Learning Rate Scheduling (Step decay, Exponential decay, Cosine annealing, Cyclical LR, Warmup)

    Handling Overfitting (Early Stopping, Data Augmentation, Label Smoothing)

    Autoencoders (Undercomplete, denoising, variational)

    Graph Neural Networks (GCN, GAT, GraphSAGE)

    Attention and Transformers (Self-attention, multi-head attention, positional encoding)
