## Directory: Computer_Vision
## Topics that must be covered
- Image Fundamentals: Pixels, Color spaces (RGB, HSV, Grayscale), Histograms
- Image Processing: Filters (Gaussian, Sobel edge detection), Morphological operations (Erosion, Dilation)
- Image Augmentation: Geometric (Rotation, Flip, Crop, Zoom) and Photometric (Brightness, Contrast, Cutout, MixUp)
- Convolutional Neural Networks (CNNs): Convolution operation (kernel, stride, padding), Pooling (Max, Average, Global)
- Classic CNN Architectures: LeNet-5, AlexNet (dropout/ReLU breakthrough), VGGNet (small filters)
- Modern CNN Architectures: ResNet (Skip connections / Residual blocks), Inception (1x1 convolutions), EfficientNet (compound scaling), MobileNet (depthwise separable)
- Object Detection (Two-stage): R-CNN, Fast R-CNN, Faster R-CNN (Region Proposal Network)
- Object Detection (One-stage): YOLO (v1 to v9), SSD, RetinaNet (Focal Loss)
- Semantic Segmentation: FCN (Fully Convolutional), U-Net (skip connections for localization), Mask R-CNN (instance segmentation), DeepLab (atrous convolution)
- Transfer Learning: Feature extraction vs. Fine-tuning, using pre-trained ImageNet weights (ResNet, ViT)
- Generative CV: Autoencoders, Variational Autoencoders (VAEs), GANs (DCGAN, StyleGAN, CycleGAN), Diffusion Models (DDPM)
