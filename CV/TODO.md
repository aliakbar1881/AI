Topics that must be covered

    What will be covered in this journey? (Complete Computer Vision pipeline from pixels to understanding)

    Image Fundamentals (Pixels, color spaces (RGB, HSV, Grayscale), histograms)

    Image Processing (Filters — Gaussian, Sobel edge detection, morphological operations)

    Image Augmentation (Geometric — rotation, flip, crop; Photometric — brightness, contrast, Cutout, MixUp)

    Convolutional Neural Networks (Convolution operation, kernel, stride, padding, pooling)

    Classic CNN Architectures (LeNet-5, AlexNet, VGGNet)

    Modern CNN Architectures (ResNet — skip connections, Inception, EfficientNet, MobileNet)

    Transfer Learning (Feature extraction vs. fine-tuning, pre-trained weights)

    Object Detection — Two-stage (R-CNN, Fast R-CNN, Faster R-CNN)

    Object Detection — One-stage (YOLO v1–v9, SSD, RetinaNet with Focal Loss)

    Semantic Segmentation (FCN, U-Net, Mask R-CNN, DeepLab with atrous convolution)

    Generative CV (Autoencoders, VAEs, GANs — DCGAN, StyleGAN, CycleGAN, Diffusion Models)

    Vision Transformers (ViT, DETR, Swin Transformer)

    Multi-modal Vision (CLIP, BLIP, image captioning, VQA)
